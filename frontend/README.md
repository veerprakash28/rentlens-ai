# 🖼️ RentLens Frontend

This is the frontend for the **RentLens** project, built with [Next.js](https://nextjs.org) and styled with [TailwindCSS](https://tailwindcss.com). It provides a clean, modern interface for interacting with the Facebook Group scraper and viewing results parsed by Gemini AI.

---

## 🚀 Getting Started

First, ensure you have the backend running (refer to the root [README](../README.md)).

Then, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the results.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org) (App Router)
- **Styling**: [TailwindCSS 4](https://tailwindcss.com)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **AI Integration**: Directly uses the Gemini SDK for client-side tasks where appropriate.

---

## 📂 Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/`: Reusable UI components.
- `lib/`: Utility functions and API clients.
- `public/`: Static assets.
